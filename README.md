# Primer Proyecto React

Este proyecto es una práctica inicial para aprender los fundamentos de React utilizando Vite como herramienta de desarrollo. El objetivo es familiarizarse con la estructura de un proyecto moderno, el uso de componentes y el flujo básico de trabajo en React.

## Tecnologías utilizadas
- React
- TypeScript
- Vite
- ESLint
