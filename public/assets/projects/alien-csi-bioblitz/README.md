# Imágenes - Alien CSI Bioblitz

Coloca aquí las imágenes para el case study de Alien CSI Bioblitz.

## Estructura sugerida:

- `hero-image.jpg` - Imagen principal para la sección hero
- `fase-1-evento.jpg` - Imagen de la fase 1 (Pre-evento)
- `fase-2-durante.jpg` - Imagen de la fase 2 (Durante evento)
- `fase-3-post.jpg` - Imagen de la fase 3 (Post-evento)
- `my-role.jpg` - Imagen para la sección "Mi Rol"

## Uso en el código:

```tsx
import heroImage from '../assets/projects/alien-csi-bioblitz/hero-image.jpg';

// En la sección:
<img src={heroImage} alt="Descripción" />
```
